# Artificial Neural Network Deep Learning Projects

## Overview

This repository contains multiple Machine Learning and Deep Learning projects implemented using Python. These projects demonstrate practical applications of advanced algorithms such as Restricted Boltzmann Machines (RBM) and Long Short-Term Memory (LSTM) networks.

---

## Projects Included

### 1. Customer Segmentation using RBM

* Unsupervised learning technique
* Identifies customer groups based on behavior
* Helps in targeted marketing strategies

### 2. Stock Price Prediction using LSTM

* Time-series forecasting using deep learning
* Predicts stock price trends based on historical data
* Uses LSTM (Recurrent Neural Network)
