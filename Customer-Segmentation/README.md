# Customer Segmentation using RBM

## Project Description

This project focuses on customer segmentation using an unsupervised learning technique called Restricted Boltzmann Machine (RBM).

The goal is to group customers based on their purchasing behavior and patterns, enabling businesses to design targeted marketing strategies.

---

## Data

* Dataset: Customer transaction or behavioral dataset
* Type: Unsupervised data (no labels)
* Features may include:

  * Customer ID
  * Purchase history
  * Spending patterns

---

## Technology Stack

* Programming Language: Python
* Libraries:

  * NumPy
  * Pandas
  * Matplotlib
  * Seaborn
  * scikit-learn

---

##  Model Used

* Restricted Boltzmann Machine (RBM)
* Unsupervised Learning
* Feature learning and pattern discovery

---

## Steps Involved

1. Data Loading
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Model Training using RBM
5. Customer Segmentation
6. Visualization of clusters

---

## Results

* Customers are grouped into meaningful segments
* Helps identify:

  * High-value customers
  * Low engagement users
  * Potential target groups

---

## Applications

* Marketing strategy optimization
* Personalized recommendations
* Customer retention analysis
