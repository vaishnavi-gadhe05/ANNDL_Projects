# Stock Price Prediction using LSTM

## Project Description

In this project, we predict stock prices using a deep learning model based on Recurrent Neural Networks (RNNs) with Long Short-Term Memory (LSTM).

It is based on the historical stock price data of Apple Inc. (AAPL).

---

## Data

* Source: Yahoo Finance
* Stock: Apple Inc. (AAPL)
* Time Range: Jan 2019 - Dec 2024

### Features:

* Open
* High
* Low
* Close
* Adj Close
* Volume

---

## Technology Stack

* Programming Language: Python
* Libraries:

  * NumPy
  * Pandas
  * Matplotlib
  * Seaborn
  * scikit-learn
  * TensorFlow / Keras
* API: yFinance

---

## Deep Learning Model

* Recurrent Neural Network (RNN)
* LSTM (Long Short-Term Memory)
* Dropout Layers
* Early Stopping

---

## Steps Involved

1. Data Extraction via yFinance API
2. Data Preprocessing
3. Splitting Training and Testing Data
4. Developing LSTM Model
5. Model Training
6. Prediction & Evaluation
7. Result Visualization

---

## Performance Measures

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)

---

## Execution Process

1. Open the notebook in Google Colab or Jupyter Notebook
2. Install required dependencies
3. Run all cells sequentially
4. Observe predictions and visualizations

---

## Results

The model learns patterns from historical stock data and provides predictions that closely follow actual trends.

---

## Conclusion

The LSTM model successfully captures time-dependent patterns in stock price movements. However, due to market volatility, predictions may vary and should be used cautiously.
